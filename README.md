# Vendor Management System

A full-stack application built using React, Spring Boot, and MySQL for managing vendors and their approval workflow.

## Features

- Add Vendor
- View Vendors
- Approve Vendor
- Reject Vendor
- Delete Vendor
- Dashboard Analytics

## Tech Stack

- React.js
- Spring Boot
- MySQL
- Axios
- REST API
